Installation
============

Add the package as dependency to your setup.py:

.. code:: python

  setup(...
        install_requires=[
          ...
          'ftw.theming',
        ])

or to your buildout configuration:

.. code:: ini

  [instance]
  eggs += ftw.theming

and rerun buildout.
